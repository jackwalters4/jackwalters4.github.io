BundL iOS Mobile App

My friend and I built all native functionality for the BundL mobile app

With the beta app, users can:

  • Sign up for their BundL card
  • Add funds to their BundL card
  • Get notifications of their Cashback earned
  • View transaction history

Tech Stack

  • Swift for native iOS development
  • Firebase for user auth and realtime database
  • Apto Payments API for card issuing and KYC process
